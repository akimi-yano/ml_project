Technologies

  • Python: Main language used for this project
  • Django: Framework used to put together this website
  • Open CV: Heavily used for computer vision processing for images and videos
  • Numpy: I needed to use NumPy since OpenCV represents images as multi-dimensional NumPy arrays
  • Dlib: The dlib library includes the facial landmark detector which is an implementation of the "One Millisecond Face Alignment with an Ensemble of Regression Trees" paper by Kazemi and Sullivan (2014)
  • Imutils: The imutils library includes face_utils that contains mappings of the facial landmark indexes
  • Pillow: Python Imaging Library (abbreviated as PIL) (in newer versions known as Pillow) is a free library for the Python programming language that adds support for opening, manipulating, and saving many different image file formats
  • Pip: Used to manage the environment created for this project
  • Haar cascades: Machine learning object detection algorithm used to identify objects in an image or video and based on the concept of features proposed by Paul Viola and Michael Jones in their paper "Rapid Object Detection using a Boosted Cascade of Simple Features" in 2001
  • GitHub: Used to import modules and read documentations
  • JavaSctipt, HTML and CSS: Used for front-end designing
  • Open CV Tutorials: Inspired and learned from tutorials provided by Adrian Rosebrock
